# C语言判断语句

C语言把任何非零和非空的值作为true，把零或null作为false。

判断语句的几种形式：

1. if语句

```c
if(boolean_expression){
  // statement
}else if{
  // statement
}else{
  // statement
}
```

2. swith语句

```c
swich(expression){
  case constant_expression:
    // statement
    break;
  case constant_expression:
    // statement
  default:
    // statement
}
```

3. 三元运算符

```c
(num % 2 == 0) ? printf("偶数") : printf("奇数");
sum = char_value == 'A' ? A * number : B * number;
```
