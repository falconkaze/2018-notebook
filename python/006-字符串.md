# python字符串使用

## 介绍

在python中，用引号（单引号或双引号）括起的都是字符串。python中没有字符类型（char），单个字符的字符串就相当于char值。

## 使用

### 方法

#### 大小写转换

`title` 以首字母大写的方式显示每个单词，除了首字母之外的字符设置为小写
`lower()` 将字符串转为小写
`upper()` 将字符串转为大写

#### 合并字符串

* 使用`+`来拼接字符串。在python中，数字和字符串不能直接拼接，需要先用`str`函数把数字转为字符串。

#### 删除空白

`trip()` 删除字符串两端的空白
`ltrip()` 删除字符串左边的空白
`rtrip()` 删除字符串右端的空白
