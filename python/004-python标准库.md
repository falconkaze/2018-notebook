## 日志模块（`logging`）

```
logging.basicConfig(
  level=logging.DEBUG,
  format='%(asctime)s : %(levelname)s : %(message)s',
  filename=logging_file,
  filemode='w',
)
logging.debug("Start of the program")
logging.info("Doing something")
logging.warning("Dying now")
```
