# google-chrome使用

## 快捷键

| 打开新窗口                     | Ctrl+n                    |
| 在无痕模式下打开新窗口         | Ctrl+shift+n              |
| 打开新标签页并跳转过去         | Ctrl+t                    |
| 打开上一个关闭的标签页并跳转   | Ctrl+shift+t              |
| 跳转到上一个标签页             | Ctrl+shift+Tab或Ctrl+PgUp |
| 跳转到下一个标签页             | Ctrl+Tab或Ctrl+PgDn       |
| 打开当前标签页记录的上一个页面 | Ctrl+左箭头               |
| 打开当前标签页记录的下一个页面 | Ctrl+右箭头               |
| 关闭当前标签页                 | Ctrl+w                    |
