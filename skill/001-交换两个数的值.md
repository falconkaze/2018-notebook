# 不使用额外空间交换两个数的值

核心就是所采用的运算要保证运算是可逆的，或者说运算不会导致信息丢失。具体来说，加法可以根据和和一个操作数来计算出另一个操作数，运算可逆；异或根据结果和一个操作数可以推算出另一个操作数（1 ^ a = 1，那么a一定为0）。从这个角度来说，用乘法也可以实现，但是乘法会导致出现浮点数，会有精度问题，另一方面，乘法的效率要低很多。

1. 利用按位异或`^`

C语言实例：

```c
unsigned int a = 60;
unsigned int b = 13;
a = a ^ b;
b = a ^ b;
a = a ^ b;
```

2. 利用加法

C语言实例：

```c
unsigned int a = 60;
unsigned int b = 13;
a = a + b;
b = a - b;
a = a - b;
```
